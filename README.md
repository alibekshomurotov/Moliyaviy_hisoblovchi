# 📊 Shaxsiy Moliya Dashboard

## 🎯 Loyiha haqida

Bu loyiha **Shaxsiy Moliya va Analytics Dashboard** - foydalanuvchilarning kirim va chiqimlarini kuzatish, tahlil qilish va moliyaviy maslahatlar olish imkoniyatini beruvchi veb-ilova.

## ✨ Asosiy xususiyatlar

### 1. 📊 Tranzaksiyalarni boshqarish
- Kirim va chiqimlarni qo'shish
- Tranzaksiyalarni tahrirlash va o'chirish
- Kategoriyalar bo'yicha saralash (Oziq-ovqat, Transport, Kommunal va boshqalar)

### 2. 📈 Analytics va Grafiklar
- Kategoriyalar bo'yicha harajatlar grafigi (Pie Chart)
- Oylik taqqoslash grafigi (Bar Chart)
- Recharts kutubxonasi yordamida vizualizatsiya

### 3. 🤖 AI Yordamchi (Sun'iy Intellekt)
- Moliyaviy savollarga javob beradi
- Tejamkorlik bo'yicha maslahatlar
- Balans va statistikani ko'rsatadi
- OpenRouter API orqali real AI (yoki offline rejim)

### 4. 🌓 Dark / Light Mode
- Ikki xil tema
- Foydalanuvchi tanlovi saqlanadi

### 5. 💾 Local Storage
- Barcha ma'lumotlar brauzerda saqlanadi
- Internet yo'q bo'lganda ham ishlaydi

### 6. 📅 Filtrlash
- Sana bo'yicha filtrlash
- Kirim/chiqim bo'yicha filtrlash
- JSON formatda Import/Export

### 7. 💱 Valyuta va Til
- 3 ta valyuta: UZS, USD, RUB
- 3 ta til: O'zbek, English, Русский
